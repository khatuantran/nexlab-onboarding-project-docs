# CR-003 — Free-tier pilot deployment (replace VPS path of M3)

## Summary

Thay đường VPS + Docker Compose + Nginx + Let's Encrypt của M3 bằng stack free-tier managed (Cloudflare Pages + Fly.io + Neon Postgres + Upstash Redis) để pilot không tốn chi phí.

## Motivation

- **Pilot scale nhỏ** (< 50 user, < 100 feature) — không justify chi phí VPS + DBaaS.
- **Đường VPS** trong [roadmap.md M3](../roadmap.md) yêu cầu ops effort (config Nginx, Let's Encrypt cron, backup, monitoring, OS update) → mất thời gian không tạo giá trị cho pilot scope.
- **Free-tier managed** đẩy ops cho provider; team chỉ tập trung vào product validation.
- **Teardown dễ**: nếu pilot thất bại, xoá projects trên Cloudflare/Fly/Neon/Upstash trong vài phút thay vì decommission VPS.
- User feedback 2026-04-26: cần $0 budget cho pilot.

## Proposed change

### Before

[roadmap.md M3](../roadmap.md) deliverable:

- VPS (DO Droplet / Hetzner) chạy Docker Compose prod config.
- Nginx reverse proxy + Let's Encrypt cert.
- Managed Postgres + Redis (hoặc self-host).
- Manual `pg_dump` cron weekly.

### After

| Layer                        | Provider                                       | Free quota                                       | Region               |
| ---------------------------- | ---------------------------------------------- | ------------------------------------------------ | -------------------- |
| FE (SPA)                     | Cloudflare Pages                               | Unlimited bandwidth, 500 builds/mo               | global edge          |
| BE                           | Fly.io                                         | 3 shared-cpu-1x VM 256MB + 3GB volume            | sin                  |
| Postgres                     | Neon                                           | 0.5GB storage, 100 CU-hours/mo, autosuspend 5min | sin (ap-southeast-1) |
| Redis (session + rate limit) | Upstash                                        | 10k cmd/day, 256MB                               | ap-southeast-1       |
| File uploads                 | Fly persistent volume                          | 3GB                                              | sin                  |
| CI/CD                        | GitHub Actions (private repo)                  | 2000 min/mo                                      | —                    |
| Monitoring                   | Fly metrics + Cloudflare Analytics + pino logs | built-in                                         | —                    |

URLs (no custom domain v1): `https://<project>.pages.dev` (FE) + `https://onboarding-api.fly.dev` (BE).

Backup: Neon PITR free 7 days + manual `fly ssh -C "pg_dump"` weekly into Cloudflare R2 (defer setup).
