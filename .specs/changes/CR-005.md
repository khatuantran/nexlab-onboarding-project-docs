# CR-005 — UI Quality Uplift v2 (geometric patterns + multi-accent palette)

<!-- template: 05-change-request-template.md@0.1 -->

- **Status**: Proposed
- **Requester**: @khatrannexlab

- **Decided at**: 2026-05-15 (sẽ update khi user approve pilot)

## Summary

Charter v3 amendment lift illustration ban — cho phép **geometric SVG patterns** (blob, grid, gradient mesh, wave, dot pattern) + mở rộng **palette accent** (blue / green / purple / pink / cyan / amber) + giảm text density ở shell / hero / empty / landing. Triển khai pilot-first qua HomePage trước, roll out 3 màn (Login, ProjectLanding, Search) sau review.

## Motivation

User feedback 2026-05-15: ngôn ngữ thiết kế hiện tại "quá ít màu, quá nhiều chữ". CR-002 ship v1 charter (orange + gold + grayscale, hero gradient + NxLogo mark only) đã thiết lập rich tokens nhưng:

- Toàn product chỉ 2 brand color → mọi screen visually đồng nhất, thiếu category identity.
- Charter v2 §7 cấm illustration ("no stock illustrations / unDraw in v1") → hero / empty / landing thiếu visual reward, phụ thuộc nhiều vào copy.
- Hero panels có 3-4 dòng subtitle copy → noise, đặc biệt mobile.

User-confirmed direction:

1. **Visual**: Geometric shapes & patterns (Stripe / Linear style — abstract SVG + CSS gradients). KHÔNG character illustrations (Storyset / unDraw).
2. **Color**: Mở rộng palette accent (+4-6 màu) cho category / section / status; orange + gold giữ vai trò brand + primary CTA.
3. **Text reduction**: Chỉ cắt copy ở shell (nav / header), hero, empty states, landing. **Giữ nguyên** content body (feature-detail 5-section prose là core value của product).
4. **Tone**: Warmer professional (Stripe / Linear / Vercel feel) — không playful, không bold expressive.

Charter v2 đã thiết lập momentum visual richness; CR-005 đẩy thêm 1 bậc theo direction user lock.

## Proposed change

### Before

- `.specs/ui/visual-language.md` v2 §7 lock _"No stock illustrations / unDraw / Storyset in v1 — too generic. Empty states use lucide icon only."_
- Palette: orange `--primary` + gold `--secondary` + neutral surfaces + semantic (success / warning / info / destructive). Không có "accent" group cho category identity.
- Empty states: lucide icon 64×64 `text-primary/40`, không decoration.
- Hero panels: gradient backdrop từ `primary-50` → `background` → `secondary-bg/50` + optional NxLogo mark; subtitle 2-3 dòng copy.
- Code: `apps/web/src/components/patterns/` directory không tồn tại.

### After

- Charter v3 §7 rewrite: ✅ allow geometric SVG patterns (blob, grid, gradient mesh, wave, dot pattern) + CSS gradients; ❌ still ban character illustrations / stock photography / 3D renders / hand-drawn sketches.
- Charter v3 §2 thêm "accent palette" group (info-blue / success-green / purple / pink / cyan / amber) + usage rule "1 accent per surface, accent dùng cho category / section / status, không thay primary CTA".
- Charter v3 §10 thêm pattern primitives section (BlobBackdrop / GridPattern / GradientMesh / WaveDivider / DotField).
- Charter v3 §6 amend: shell / hero / empty được phép giảm 30-50% text; content rows + detail body giữ nguyên density target.
- Design-system.md §1 thêm accent token map (HSL light + dark); §5 thêm Pattern primitives section.
- Per-screen UI specs (Phase A — 4 screens) update Visual structure:
  - **HomePage** (pilot): hero pattern backdrop + ProjectCard accent border-l theo category + reduce hero copy + empty state DotField.
  - **LoginPage**: gradient mesh + BlobBackdrop trên brand panel + tagline 1 dòng.
  - **ProjectLandingPage**: hero gradient mesh + FeatureCard section-type accent color (5 section = 5 màu).
  - **SearchPage**: DotField empty state + reduce hero copy.
- Code: new `apps/web/src/components/patterns/` directory + accent palette CSS vars + per-screen page refactor.
