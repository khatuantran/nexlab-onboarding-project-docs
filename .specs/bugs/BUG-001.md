# BUG-001 — `pnpm docker:*` scripts không load `.env.local`

<!-- template: 05-bug-template.md@0.1 -->

- **Severity**: 🟡 Medium
- **Priority**: P1
- **Status**: Open

- **Fixed at commit**: `<hash>` (fill when fixed)
- **Environment**: dev

## Reproduction

1. Set override trong `.env.local` ở repo root, ví dụ: `POSTGRES_PORT=5433`.
2. `pnpm docker:up`.
3. `docker compose -f infra/docker/docker-compose.yml ps` → Postgres vẫn published `0.0.0.0:5432->5432/tcp`.

Tương tự với `POSTGRES_USER` / `POSTGRES_PASSWORD` / `POSTGRES_DB` / `REDIS_PORT` — mọi override infra đều bị bỏ qua.

**Prerequisites**: `.env.local` tồn tại ở repo root với biến override.

**Frequency**: always.

## Expected vs Actual

- **Expected**: Compose đọc `.env.local` ở repo root → `POSTGRES_PORT=5433` → container bind `5433:5432`. Docs [SETUP.md §4](../../docs/SETUP.md) hướng dẫn chính xác flow này.
- **Actual**: Compose không load `.env.local`; mọi `${VAR:-default}` fallback về default.
- **Spec reference**: [SETUP.md §4](../../docs/SETUP.md) — khối hướng dẫn override port khi 5432/6379 đã bị chiếm.

## Scope

- **Affected files**: [package.json](../../package.json) (`docker:up` / `docker:down` / `docker:reset` / `docker:logs`), [.env.example](../../.env.example), [docs/SETUP.md](../../docs/SETUP.md).
- **Affected users**: dev team (local). Prod/staging không dùng compose scripts này.
- **Data corruption**: no.
- **Rollback required**: no.

## Root cause

Compose CLI với `-f <path>` đặt **project directory = thư mục chứa compose file** (ở đây `infra/docker/`), và chỉ auto-load `.env` nằm trong project directory đó. Repo chứa env ở `.env.local` (root), không phải `infra/docker/.env` → compose không thấy → `${POSTGRES_PORT:-5432}` rơi về default.

Không phải bug syntax compose-file; bug là **missing env loader flag** trong scripts.

## Fix approach

Thêm `--project-directory . --env-file .env.local` vào 4 `docker:*` scripts. Chạy compose từ repo root view với env-file explicit ở root.

Bổ sung:

- `.env.example` thêm block `POSTGRES_USER/PASSWORD/DB/PORT` + `REDIS_PORT` để dev biết override hooks tồn tại (hiện không có trong template).
- SETUP.md §4 + §11 clarify: `.env.local` phải tồn tại trước `pnpm docker:up` (decision: hard-fail, không auto-copy).

**Alternatives considered**:

- (a) Di chuyển `.env.local` → `infra/docker/.env`: rejected, tách biến API/infra thành 2 file làm tăng maintenance.
- (b) Thêm `env_file:` trong compose YAML: rejected, `env_file` inject biến vào **container environment**, không substitute `${VAR}` ở compose parse phase (vẫn không fix port binding).

**Files to change**:

- `package.json` — 4 scripts.
- `.env.example` — thêm block.
- `docs/SETUP.md` — §4 + §11.
- `scripts/test-infra-env.mjs` (new, regression harness).

**Migration needed**: no.
